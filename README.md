## ðŸ§  Model Information

### Algorithm: K-Means Clustering

**Parameters:**
- Number of clusters: 3
- Random state: 42 (for reproducibility)
- Initialization: k-means++

**Preprocessing Steps:**
1. **Currency Cleaning**: Remove $, commas, % signs
2. **Type Conversion**: Convert categorical columns to numeric
3. **Imputation**: Use median strategy for missing values
4. **Log Transformation**: Applied to skewed features
   - GDP
   - CO2 Emissions
   - Energy Usage
   - Health Exp/Capita
   - Tourism Inbound/Outbound

5. **Standardization**: StandardScaler (mean=0, std=1)

### Performance Metrics

- **Silhouette Score**: ~0.22 (normalized from -1 to 1)
- **Model Stability**: All countries classified (no noise points)
- **Training Data**: 190+ countries

### Cluster Characteristics

## ðŸ› Troubleshooting

### Model file not found
```
Error: Model file not found: world_development_kmeans.pkl
```
**Solution:** Run `python train_model.py` first

### Data loading error
```
Error: World_development_mesurement.xlsx not found
```
**Solution:** Ensure the Excel file is in the project directory

### Module import error
```
ModuleNotFoundError: No module named 'streamlit'
```
**Solution:** Install requirements: `pip install -r requirements.txt`

### Port already in use
```
ERROR: Port 8501 is already in use
```
**Solution:** Use a different port: `streamlit run streamlit_app.py --server.port 8502`
